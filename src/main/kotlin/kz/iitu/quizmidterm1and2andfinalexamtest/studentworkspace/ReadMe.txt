===== Instructions =====

Welcome to the Student Workspace.

Your Task:
1. Open the `StudentController.kt` file in this folder.
2. Complete the following tasks:
   a. Add the @RestController annotation to the class.
   b. Create a GET mapping for the endpoint "/api/test".
   c. Make the endpoint return the string "Hello from the student!".

How to Test:
1. Save your changes in `StudentController.kt`.
2. Run the test framework using the command:
    gradlew run
3. Follow the prompts in the console to execute the tests.

Note:
- You can run `test-1` to check the presence of @RestController.
- You can run `test-2` to validate the GET mapping.
